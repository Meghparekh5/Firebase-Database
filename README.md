# 📦 Inventory Tracking System

A modern Inventory Tracking System built using **React.js**, **Redux Toolkit**, and **Firebase Realtime Database**. This application allows users to manage products efficiently with real-time updates and a clean, responsive UI.

---

## 🚀 Features

* Add new products with name, category, quantity, and price
* View all products in a structured table
* Update existing product details with inline editing
* Delete products instantly
* Real-time synchronization with Firebase
* Form validation for required fields
* Clean brown-themed responsive UI

---

## 🛠 Tech Stack

* **Frontend:** React.js
* **State Management:** Redux Toolkit
* **Backend:** Firebase Realtime Database
* **Async Handling:** Redux Thunk
* **Styling:** CSS

## 🔥 Firebase Setup

1. Create a project in Firebase
2. Enable Realtime Database
3. Copy your Firebase config
4. Paste it inside:

```
src/firebase/firebase.js
```

---

## 💡 Usage

* Fill the form to add a product
* Click **Update** to edit product details
* Click **Delete** to remove a product
* All changes update instantly across UI
